# Go users

clone it into something like `gocode/src/github.com/jasn/hashcode2020`

    cd src
    go build .
    ./src a > ../solutions/a.out
    ./src b > ../solutions/b.out
    ./src c > ../solutions/c.out
    ./src d > ../solutions/d.out
    ./src e > ../solutions/e.out
    ./src f > ../solutions/f.out
